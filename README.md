# GIG 24 App

 The GIG 24 is the one stop shop for PA and film production jobs. Search for gigs and submit your resume today! 


* * *

## LINK TO LIVE APP

https://parent-connect-app.now.sh/

## TECHNOLOGIES USED

* React.js 
* CSS 
* Node.js 
* Express 
* PostgreSQL
* SQL
